# GLaDOS 自动签到插件

一个浏览器扩展（支持 **Edge** / Chrome），每天在设定时间自动打开 GLaDOS 签到页并点击签到按钮。

## 你需要准备什么

- 电脑上已安装 **Microsoft Edge**（或 Chrome，扩展通用）
- 你的 GLaDOS 账号已经能在浏览器里正常登录

## 安装步骤

### 第一步：确认文件都在

打开文件夹 `glados-check-auto`，确认里面有这些文件：

- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`
- `README.md`（本说明）

### 第二步：在 Edge 里加载扩展

1. 打开 **Edge**，在地址栏输入：**`edge://extensions/`**，回车。
2. 打开左下角的 **「开发人员模式」** 开关。
3. 点击 **「加载解压缩的扩展」**。
4. 在弹出的窗口里选中你的 **`glados-check-auto`** 文件夹（就是包含 `manifest.json` 的那个文件夹），点 **「选择文件夹」**。
5. 如果成功，你会看到列表里多出一个叫 **「GLaDOS 自动签到」** 的扩展。

> 如果用 Chrome：地址栏输入 `chrome://extensions/`，打开「开发者模式」→「加载已解压的扩展程序」，选同一文件夹即可。

### 第三步：先登录 GLaDOS

1. 在浏览器里打开：https://glados.cloud/console/checkin  
2. 如果没登录，先登录你的 GLaDOS 账号。  
3. 登录后能正常看到签到页面即可。  
4. **重要**：以后每天自动签到时，请保持 **Edge** 处于已登录状态（不要退出 GLaDOS 登录）。

### 第四步：设置每天签到时间

1. 点击 Edge 右上角 **扩展图标**（拼图块），找到 **「GLaDOS 自动签到」**，点击它。
2. 在弹出的小窗口里设置 **每天几点几分** 自动签到（例如 8 时 0 分 = 早上 8 点）。
3. 点击 **「保存时间」**，会提示“已保存”。

### 第五步：试一下“立即签到”

1. 在扩展弹窗里点击 **「立即签到」**。
2. 会自动打开签到页面，扩展会在页面上自动点击签到按钮。
3. 如果成功，说明插件工作正常；如果没点到，可能是页面改版，可以告诉我按钮长什么样，我帮你改代码。

## 每天自动运行说明

- 到你在扩展里设置的 **每天的那个时间**，Edge 会自动打开一个签到页的标签页。
- 页面加载完成后，扩展会自动在页面上 **点击签到按钮**，完成签到。
- **注意**：  
  - 到点时 **Edge 需要是打开的**（可以最小化，但不要完全退出 Edge）。  
  - 你的 GLaDOS 要在浏览器里处于 **已登录** 状态，否则打开的是登录页，无法自动签到。

## 常见问题

**Q：到时间了没有自动打开页面？**  
- 检查是否打开了 Edge（没有完全退出）。  
- 检查扩展是否已启用（在 `edge://extensions/` 里看“GLaDOS 自动签到”是否开启）。  
- 重新打开扩展弹窗，改一下时间再点一次“保存时间”，再观察第二天是否执行。

**Q：打开了页面但没有自动点击签到？**  
- 先确认你是否已经登录 GLaDOS；未登录时打开的是登录页，没有签到按钮。  
- 若已登录仍不点击，可能是网站改版导致按钮文字或结构变了，可以把当前签到按钮上的文字发给我，我帮你改选择器。

**Q：我是编程小白，不会改代码怎么办？**  
- 安装和使用按上面步骤即可，不需要改代码。若网站改版导致点击失效，把签到按钮的截图或文字发给我，我可以给你改好后的代码片段。

## 文件说明（给想看的同学）

- `manifest.json`：扩展的配置，告诉浏览器这是啥、要什么权限、什么时候跑哪个脚本。  
- `background.js`：负责“每天到点”和“打开签到页”。  
- `content.js`：在签到页里运行，负责找到签到按钮并点击。  
- `popup.html` / `popup.js`：扩展小弹窗的界面和逻辑，用来设置时间和点“立即签到”。
